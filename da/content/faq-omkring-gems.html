FAQ omkring GEMS

Jeg har glemt mit USER ID - hvad gør jeg?
Svar: Tjek din mail indboks for den første mail, du tidligere har modtaget fra AWG (volunteer administration). Her vil USER ID fremgå. Kan du ikke finde dit USER ID skal du skrive til volunteer@awg2016.gl eller ringe på 382034.

Jeg har ikke modtaget mit USER ID og Password – hvad gør jeg?
Svar: Der kan være sket fejl i systemet, så du ikke har modtaget dit USER ID og password. Kontakt volunteer@awg2016.gl eller ring på 382034.

Jeg har glemt mit password – hvad gør jeg?
Svar: Gå til GEMS login-siden. Under USER ID og Password klikker du på ’I forgot my password’. Du vil nu modtage en mail med oplysninger om, hvad du skal gøre.
 
For spørgsmål, skriv til volunteer@awg2016.gl eller ring på 382034.
